HTML

  • The head element of html contains metadata for the page, which is browser relevent and helps with SEO (search engine optimization)
  • The body element contains what is visible to the webpage user
  • It's best practice to use semantic elements to keep code easy to understand
  • HTML is the content of the webpage, the style comes with CSS, and the functionality comes with JavaScript
  • More on attributes later

CSS

  • CSS notes go here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

  • Classes and elements can be collectively styled in your CSS code
  • Add a period before class changes in the CSS sheet, elements dont need it.
